Return a Bartlett window.

The Bartlett window is very similar to a triangular window, except that the end points are at zero. It is often used in signal processing for tapering a signal, without generating too much ripple in the frequency domain.

Return a Blackman window.

The Blackman window is a taper formed by using the first three terms of a summation of cosines. It was designed to have close to the minimal leakage possible. It is close to optimal, only slightly worse than a Kaiser window.

Return an exponential (or Poisson) window.

Template Parameters
TType of returned values
Parameters
centerParameter defining the center location of the window function. The default value if not given is center = (M-1) / 2. This parameter must take its default value for symmetric windows.
NNumber of points in the output window. If zero or less, an empty array is returned.
tauParameter defining the decay. For center = 0 use tau = -(M-1) / ln(x) if x is the fraction of the window remaining at the end.
symWhen true (default), generates a symmetric window, for use in filter design. When false, generates a periodic window, for use in spectral analysis.
Returns
The window, with the maximum value normalized to 1 (though the value 1 does not appear if M is even and sym is true).
