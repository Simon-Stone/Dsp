Find the fastest convolution/correlation method. This primarily exists to be called during the method = \textquotesingle{}auto\textquotesingle{} option in convolve and correlate. It can also be used to determine the value of method for many different convolutions of the same length. In addition, it supports timing the convolution to adapt the value of method to a particular set of inputs and/or hardware.
